# sls-convert-pdf2img
convert pdf to image on aws lambda function with S3 events.

## Build.

```
$ npm i
```

## Command Memo.

```
# install
$ npm i -g serverless

# version
$ npx sls -v
or
$ npx serverless -v

# create project for nodejs
$ npx sls create -t aws-nodejs -p my-service

# offline start
$ npx sls offline start

# deploy 
$ npx sls deploy -v

# delete project
$ npx sls remove

```
